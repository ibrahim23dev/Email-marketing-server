# Apify Email Scraper API (Node.js + TypeScript + Express + MongoDB)

## Requirements
- Node.js 18+ (recommended)
- npm
- MongoDB running locally or remote
- Apify account + API token

## Quick start
1. Clone this repo and `cd` into it.
